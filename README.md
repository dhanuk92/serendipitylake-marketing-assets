# Serendipity Lake — Marketing Assets

Curated public mirror of Serendipity Lake's hero marketing photos, for use with tools (e.g. Canva) that require a publicly fetchable URL.

This is a **deliberately narrow subset** — the 17 "TOP PICK" images identified in the private production repo's `HOTEL_PHOTOS_REFERENCE.md`. It exists solely so design-automation tools can pull these specific, already-intended-for-marketing images without needing authenticated access. No other property, guest, or business content lives here.

Source of truth for the property and its content programs is private; this repo is a publish target only.

Images are in `photos/`, filenames unchanged from the source catalogue.
